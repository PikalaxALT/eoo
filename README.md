# EOO

This is a decompilation of the Wii connection utility found in the filesystem of Pokémon Diamond and Pearl.

It builds the following ROMs:

* **eoo.nds** `sha1: 0c76e361f805577c41c80de0821ce83d8602c762`

To set up the repository, see [INSTALL.md](INSTALL.md).

For contributor instructions, see [CONTRIBUTING.md](CONTRIBUTING.md).
